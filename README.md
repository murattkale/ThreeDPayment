## Kurulum

Install the [ThreeDPayment NuGet Package](https://www.nuget.org/packages/ThreeDPayment).

### Package Manager Console

```
Install-Package ThreeDPayment
```

### .NET Core CLI

```
dotnet add package ThreeDPayment
```
